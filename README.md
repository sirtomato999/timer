# timer 1.0.0
A cubing timer   
timer is a cubing timer made by me.         
Keybindings:   
0-9: Change session   
d: Delete last timer   
r: Delete all times in session (reset)         
Configuration:   
```
0:title   
1:title   
...   
...   
```

0-9: session titles   
Remember: NO SPACE AFTER COLON
